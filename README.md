# Differential Gaussian Rasterization

This is the rasterizer of "AdR-Gaussian: Accelerating Gaussian Splatting with Adaptive Radius", 
which proposes early culling and load balancing to accelerate the rendering speed of 3DGS.

More Details: [AdR-Gaussian: Accelerating Gaussian Splatting with Adaptive Radius](https://hiroxzwang.github.io/publications/adrgaussian/)
